# Mike Sperduti Website Clone

This project is a modern, responsive recreation of the [Mike Sperduti](https://www.mikesperduti.com) website. 
It features premium UI components, Unsplash high-resolution template imagery, and CSS grid layouts to showcase identical sections from the original site.

## Stack
- HTML5
- CSS3 (Vanilla, custom variables, Flexbox, layout grids)
- FontAwesome Icons

## Deployment
This project is intended to be deployed to GitHub Pages natively on the `main` branch.
